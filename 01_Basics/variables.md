

Variable:
- stores value in memory
-Values can change during execution

Identifier:
- name of variable

Example:
a = 10
b = 20

Notes: All inputs given by the user are stored in variables in memory.
name = input("Enter name: ")

Literals:
Literals are constant values assigned to variables in a program.
Ex:-a = 10                         name="Aman"
a->Is Variable                  name->Variable
10->Integer Literals        Aman->String Literals
 
Types Of Literals:
10        → Integer literal
10.5      → Float literal
"Aman"    → String literal
True      → Boolean literal
[1,2,3]   → Collection literal

Notes: Literals are stored in variables
                 Everything is an object.
Ex: a=10 
👉 Behind the scenes:
	• 10 → object 
	• a → reference (name) jo us object ko point karta hai

Rules for Variables (Identifiers):
1. Must start with letter (a-z, A-Z) or underscore (_)
2. Cannot start with number
3. Only letters, digits, underscore allowed
4. No special symbols (@, $, %, etc.)
5. No spaces allowed
6. Keywords cannot be used as variable names (if, for, else, etc.)
7. Variable names are case-sensitive (age ≠ Age)


Examples:
a = 10        # valid
_name = "ok"  # valid
a1 = 5        # valid

1a = 10       # invalid ❌
emp sal = 5   # invalid ❌ (space)
emp$sal = 5   # invalid ❌ (symbol)
Emp_aman=10 #Valid


if = 10      # invalid ❌ (keyword)
_for = 5     # valid ✔

age = 20
Age = 30     # different variables ✔
